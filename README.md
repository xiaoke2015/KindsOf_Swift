### [Build Settings](./KindsOf_Swift.xcodeproj)

- User Script Sandboxing 设置为 NO

User Script Sandboxing 设置为 NO 。这个设置控制了 Xcode 脚本的沙盒限制，默认情况下，Xcode 会将脚本放入沙盒环境中，限制其访问文件系统的权限，尤其是对某些目录（例如项目文件夹之外的文件）进行修改时，会出现权限问题。
